# MTF Analysis Reports

This directory contains Multi-Timeframe Analysis reports for various trading pairs.

---

## Report Types

### 1. Illustrative Reports (Educational)

**Example:** `btcusd-mtf-analysis-swing-20260307.md`

These reports use **fabricated data** to demonstrate exactly how the MTF system calculates each component:
- HTF bias detection (price structure, SMA calculations)
- MTF setup identification (pullback detection, RSI analysis)
- LTF entry signals (candlestick patterns, EMA reclaim)
- Trade parameters (entry, stop, target, R:R)

**Purpose:** Educational - shows the calculation methodology step-by-step.

---

### 2. Real-Time Reports (Live Data)

**Generated by:** `scripts/generate_mtf_report.py`

These reports use **real-time market data** fetched from:
- CCXT/Kraken (crypto pairs like BTC/USDT, ETH/USDT)
- Twelve Data (metals like XAU/USD, forex pairs)

**Generate a real report:**
```bash
# Usage: python scripts/generate_mtf_report.py <PAIR> <STYLE>

# Examples:
python scripts/generate_mtf_report.py BTC/USDT SWING
python scripts/generate_mtf_report.py ETH/USDT INTRADAY
python scripts/generate_mtf_report.py XAU/USD DAY
```

**Output:** `docs/reports/{pair}-mtf-analysis-{style}-{date}.md`

## Report Sections

1. **Executive Summary** - Signal, alignment score, entry/stop/target
2. **HTF Analysis** - Directional bias (50/200 SMA, price structure)
3. **MTF Analysis** - Setup identification (pullback, divergence)
4. **LTF Analysis** - Entry signals (candlestick patterns, EMA reclaim)
5. **Alignment Scoring** - 3-timeframe alignment assessment
6. **Trade Parameters** - Entry, stop loss, target, R:R ratio
7. **Risk Warning** - Important disclaimers

## Understanding the Analysis

### Alignment Score

| Score | Quality | Meaning |
|-------|---------|---------|
| 3/3 | HIGHEST | All timeframes aligned - trade aggressively |
| 2/3 | GOOD | 2 timeframes aligned - standard risk |
| 1/3 | POOR | Only 1 timeframe aligned - avoid or reduce size |
| 0/3 | AVOID | No alignment - do not trade |

### Signal Types

- **BUY** - All indicators aligned bullish
- **SELL** - All indicators aligned bearish
- **WAIT** - Partial alignment, wait for confirmation
- **AVOID** - Conflicting signals or insufficient data

## Automation

### Generate Daily Reports

```bash
# Add to crontab for daily 8 AM reports
0 8 * * * cd /path/to/project && python scripts/generate_mtf_report.py BTC/USDT SWING
```

### Generate for Multiple Pairs

```bash
#!/bin/bash
# scripts/generate_all_reports.sh

pairs=("BTC/USDT" "ETH/USDT" "XAU/USD" "XAG/USD")
style="SWING"

for pair in "${pairs[@]}"; do
    python scripts/generate_mtf_report.py "$pair" "$style"
done
```
